Simpledit Command Documentation

Welcome to Simpledit, a console-based molecular editor for building and manipulating molecular structures interactively. This documentation provides comprehensive command references and practical test suites to help you master all features.

Getting Started

Simpledit uses a command-line interface for all operations. Commands follow a simple syntax:

command [arguments]

Most commands have shorter aliases for convenience. For example, select can be abbreviated as sel.

Basic Commands

help (alias: h)

Display available commands or help for specific command.

help              # Show all commands
help select       # Show help for select command

list (alias: ls)

List atoms in current molecule.

list              # List all atoms
list selected     # List only selected atoms

clear (alias: cls)

Clear console output.

clear

Atom Management

add (alias: a)

Add atom at specified coordinates or import from file format.

add C 0 0 0       # Add carbon at origin
add H 1.09 0 0    # Add hydrogen at position
add xyz <<EOF     # Import XYZ format
3
Water
O  0.000  0.000  0.000
H  0.757  0.586  0.000
H -0.757  0.586  0.000
EOF

select (alias: sel)

Select atoms by index, range, or wildcard.

select 0 1 2      # Select atoms 0, 1, 2
select 0:3        # Select atoms 0, 1, 2, 3 (range)
select 0 2:4 7    # Mix individual and range
select :          # Select all atoms

delete (alias: del)

Delete atoms by index, range, or wildcard.

delete 0 1        # Delete atoms 0 and 1
delete 2:5        # Delete atoms 2, 3, 4, 5
delete :          # Delete all atoms

info (alias: i)

Show atom information or geometric measurements.

info              # Show selected atoms
info 0            # Show atom 0 details
info 0 1          # Show distance
info 0 1 2        # Show angle
info 0 1 2 3      # Show dihedral

Bond Operations

bond (alias: b)

Create bond between two atoms.

bond 0 1          # Create bond between atoms 0 and 1

unbond (alias: ub)

Remove bond between two atoms.

unbond 0 1        # Remove bond

adjustbond (alias: ab)

Automatically create bonds based on distance threshold.

adjustbond        # Auto-detect and create bonds

setthreshold (alias: st)

Set distance threshold for automatic bonding.

setthreshold 1.6  # Set threshold to 1.6 Angstroms

Geometry Adjustments

setdist (alias: sd)

Set distance between two atoms.

setdist 0 1 1.5   # Set bond length to 1.5 Angstroms

setangle (alias: sa)

Set angle between three atoms.

setangle 0 1 2 120  # Set angle to 120 degrees

setdihedral (alias: sdi)

Set dihedral angle between four atoms.

setdihedral 0 1 2 3 180  # Set dihedral to 180 degrees
Important Note

Geometry commands move the molecular fragment on the "far side" of the selected atoms. The order of atom selection is critical for achieving desired results.

Display Settings

label (alias: lbl)

Control atom label display.

label -s          # Show element symbols
label -n          # Show atom numbers
label -a          # Show both
label off         # Hide labels

camera (alias: cam)

Change camera control mode.

camera orbit      # Orbit camera mode
camera trackball  # Trackball mode

projection (alias: proj)

Change projection method.

projection perspective   # Perspective projection
projection orthographic  # Orthographic projection

Fragment Analysis

fragments (alias: frags)

List all disconnected fragments.

fragments         # Show all fragments

fragment (alias: frag)

Select all atoms in fragment containing specified atom.

fragment 0        # Select fragment containing atom 0

Molecule Management

molecules (alias: mols)

List all molecules.

molecules         # Show all molecules

new

Create new molecule.

new               # Create with auto name
new "Water"       # Create with custom name

switch (alias: sw)

Switch to different molecule.

switch 0          # Switch by index
sw "Water"        # Switch by name

rename (alias: rn)

Rename current molecule.

rename "H2O"      # Rename active molecule

remove (alias: rm)

Remove molecule.

remove 1          # Remove by index
rm "Water"        # Remove by name
Independent State

Each molecule maintains its own undo/redo history and visual settings. Switching molecules automatically saves and restores these states.

Copy/Paste/Merge

copy (alias: cp)

Copy selected atoms to clipboard.

select 0 1
copy              # Copy selected atoms

paste (alias: pa)

Paste atoms from clipboard.

paste             # Paste with 5A offset

cut (alias: ct)

Cut selected atoms (copy + delete).

select 0 1
cut               # Cut atoms

merge (alias: mg)

Merge molecule into active molecule.

merge 0           # Merge molecule 0
merge "Water"     # Merge by name
Spatial Offset

Pasted atoms are automatically offset by 5 Angstroms along the x-axis to prevent overlap with existing structures. Merge operations use the same offset strategy.

Utility

time (aliases: sleep, wait)

Wait for specified seconds (for test scripts).

time 2            # Wait 2 seconds
sleep 1.5         # Wait 1.5 seconds

Range Syntax

Range syntax start:end is inclusive on both ends. For example, 0:3 selects atoms 0, 1, 2, and 3. This works with both select and delete commands.

add C 1 0 0
add C 2 0 0
add C 3 0 0
